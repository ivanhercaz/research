How to play

Very easy! You have to find the cats in the image. When you find one cat, you will see the correct license notice and check the original imagen. But don't forget to click in the cat!

There is 8 cats in the port of Santa Cruz de Tenerife, will you find them all?

About the game

This "find the cat" game has been made as task of the course Creative Commons Certification Course. You can check all the code in GitHub and reuse it.
